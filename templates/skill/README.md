# `templates/skill/`

Reference template for the `SKILL.md` file generated by
`cli-anything-skillgen`.

The actual rendering is done programmatically in
`cli_anything_skillgen::render_skill_markdown_from_metadata` (it is
not a handlebars/tera template at runtime). This file captures the
**intended shape** of the output so:

* humans can diff-review the format in one place,
* curated packages (`packages/{gimp,blender,drawio}/skills/SKILL.md`)
  can be compared against a single source of truth,
* future template engines (tera, handlebars, askama) have a starting
  point.

When the renderer in `cli-anything-skillgen` changes shape, update
this template in the same PR and regenerate the curated files with:

```bash
cargo run --example regen_skills -p cli-anything-skillgen
```

Placeholders use the `{{name}}` / `{{#each ...}}` conventions from
handlebars so the template reads naturally even though it is not
currently consumed by a template engine.
